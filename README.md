# Codex Chip Plugin Marketplace

This repository is a Codex plugin marketplace for chip development plugins.

## Marketplace

Add this marketplace in Codex with:

| Field | Value |
| --- | --- |
| 来源 | `Harth-Y/codex-chip-plugins` |
| Git 引用 | `main` |
| 稀疏路径 | leave blank |

The marketplace also keeps a Codex marketplace layout under `plugins/codex/`.
If a Codex client supports using sparse paths as marketplace roots, use
`plugins/codex`; otherwise leave sparse path blank so Codex can read the root
`.agents/plugins/marketplace.json`.

## Plugins

- `om6626`: OM6626 / OnMicro OM6XXX BLE, 2.4GHz RF, low-power, flash/NV, board bring-up, and Keil SDK development expertise.
